# 🚀 FinanzApp

**FinanzApp** es una aplicación de gestión financiera personal diseñada para ser rápida, inteligente y 100% "Offline-First". Construida con las tecnologías más modernas de la web, permite llevar un control total de gastos, ingresos, deudas y recordatorios sin depender de la nube.

## ✨ Características Principales

### 🤖 FinanzBot (NLP Inteligente)

La joya de la corona. Olvidate de llenar formularios tediosos. Registrá todo hablando naturalmente:

- **Entiende el lenguaje coloquial**: _"Gaste 20.000 en el Coto"_, _"Me ingresaron 400.000 por freelance"_, _"Me deben 5000 de Pedro"_.
- **Categorización Automática**: Mapea automáticamente palabras como "carne", "nafta", "netlfix" o "memoria ram" a sus categorías lógicas.
- **Manejo de Formatos Locales**: Soporta puntos y comas como separadores de miles de forma inteligente (ej. 240.000).

### 📊 Dashboard Dinámico

- **Visualización Vibrante**: Gráficos de torta con paletas de colores contrastantes para identificar tus gastos de un vistazo.
- **Balance Neto**: Seguimiento en tiempo real de tu salud financiera.
- **Accesos Rápidos**: Tarjetas interactivas que filtran tus transacciones automáticamente.

### 💳 Gestión de Deudas

- Control detallado de quién te debe y a quién le debés.
- Soporte para pagos parciales y visualización de progreso.

### 🔔 Recordatorios Inteligentes

- Agenda vencimientos de servicios o cobros.
- Notificaciones locales (PWA) para que no se te pase nada.

### 📱 Experiencia PWA & Offline

- **Offline-First**: Tus datos se guardan en tu dispositivo usando IndexedDB a través de Dexie.js. No necesitás internet para usar la app.
- **Instalable**: Funciona como una aplicación nativa en iOS y Android gracias a su configuración de Web Manifest.

## 🛠️ Stack Tecnológico

- **Framework**: [Next.js 15 (App Router)](https://nextjs.org/)
- **UI & Estilos**: [React 19](https://react.dev/), [Tailwind CSS 4](https://tailwindcss.com/)
- **Animaciones**: [Framer Motion](https://www.framer.com/motion/)
- **Base de Datos**: [Dexie.js](https://dexie.org/) (IndexedDB)
- **Iconos**: [Lucide React](https://lucide.dev/)
- **Tipografía**: Urbanist (vía Google Fonts)

## 🚀 Instalación y Desarrollo

1. Clonar el repositorio:

   ```bash
   git clone https://github.com/tu-usuario/finanzapp.git
   ```

2. Instalar dependencias:

   ```bash
   npm install
   ```

3. Correr el servidor de desarrollo:

   ```bash
   npm run dev
   ```

4. Abrir [http://localhost:3000](http://localhost:3000) en tu navegador.

## 🎨 Design Aesthetics

FinanzApp utiliza un sistema de diseño **"Dark Premium"** con:

- **Glassmorphism**: Superficies con desenfoque de fondo y bordes sutiles.
- **Vibrant Gradients**: Uso de verdes neón y colores eléctricos para resaltar métricas.
- **Micro-interacciones**: Feedback visual en cada clic y transiciones suaves entre páginas.
